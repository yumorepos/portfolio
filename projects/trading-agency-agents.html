Trading Agency Agents

Status: Complete

Overview

4 specialized trading agents for Polymarket prediction markets built using The Agency framework (50K+ GitHub stars). Demonstrates agent design, systematic trading logic, and rigorous testing.

Tech stack

Python, The Agency Framework (MIT License), pytest (14 tests), GitHub Actions CI/CD

How it works

  • Market Scanner: Finds arbitrage and correlation opportunities across 200+ markets in <5 seconds
  • Risk Manager: Kelly Criterion sizing, position caps (5% max), stop-loss ($20), circuit breakers (-5%, -10%, -15%)
  • Execution Agent: Places trades with precision, monitors slippage (<0.5% target), implements retry logic
  • Performance Monitor: Tracks win rate, Sharpe ratio, max drawdown, daily P&L

Why I built this

To explore multi-agent trading systems and demonstrate my ability to build modular, well-tested Python applications. This project shows I can design specialized agents with clear responsibilities, implement robust testing, and integrate with modern frameworks.

Interesting challenges

  • Agent Specialization: Designed 4 agents with distinct roles (scan, risk, execute, monitor) for separation of concerns
  • Risk Framework: Implemented Kelly Criterion for position sizing and multi-level circuit breakers for capital protection
  • Market Detection: Built arbitrage detector using (1 - (p_yes + p_no)) > 0.5% and correlation breakdown using z-score > 1.5
  • Testing Discipline: Wrote 14 unit tests covering all agent logic with 100% pass rate
  • Framework Integration: Integrated with The Agency framework (OpenClaw + Claude Code / Cursor / Copilot)
